The Command Types

There are a few ways to trigger a command, each have their own differences.

The Prefix Type

This one is only ran when the message content starts with your predefined prefix
e.g. if your prefix is ! then !help would run it

This is the only command type that has {{arg}} variables, learn more

The Contains Type

This is ran if the message content contains a specific keyword
e.g. if your looking for "hello", and I type "Hello world!", your command would run

These do NOT have {{arg}} variables

Message Sent

Ran when a message is sent.
