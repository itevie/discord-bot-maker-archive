If Statements

If statements are a crucial thing to make your bot, they are able to check if a certain condition matches a criteria, if it doesn't you can have it run some other actions instead.

For example:

If {{message:content}} is hello
 reply-to-authors-message Hello!
If {{message:content}} is bye
 reply-to-authors-message Bye!

Obviously, this won't work as it is not in action form.

But what happens here is, it first checks if the message content is "hello",
 If it is: it'll respond with "hello!" and continue to the next action
 If it isn't: it'll skip the action after the if (reply-to-authors-message Hello!) and continue after that

After this it'll do the same with the second if.

One action if's

At the moment, full ifs do not exist so one action ifs exist, these check a condition and only execute or skip the action after this
(you can see this visually as there is an indent in the action creator.)

The types

Type What it does
Equals Runs the next action only if statement 1 IS statement 2
Not Equals Runs only if statement 1 IS NOT statement 2
Starts With Runs only if statement 1 STARTS WITH statement 2
Ends With Runs only if statement 1 ENDS WITH statement 2
Includes Runs only if statement 1 CONTAINS statement 2
