How to obtain a Discord Token

First of all, make sure you have created a Discord Bot first on the Discord Developer Portal, if you haven't, go

Steps

  1. Navigate to the Discord Developer Portal, login if you have to
  2. Click on the application you had created
  3. On the sidebar, click "Bot"
  4. Find "Reset Token", click it
  5. If it asks you, verify and then copy down the token somewhere
Important

Do NOT EVER show this token to ANYONE no matter what, if you have accidentally shown it, you must immediatly reset it
Otherwise, anyone can do anything they want with your bot
