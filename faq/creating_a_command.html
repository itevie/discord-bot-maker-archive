Creating a command

Commands are a fundamental thing for your bot's functionality.

To get started, let's understand all the different parts of the bot creator.

Command Name

The command name is the identifer for every command, for example, if your command type is prefix then this will be the command that you will run
For example if your command name is "help" and your prefix is "!" then you would run it by typing "!help"

Note: a command name must be unique, 2 commands cannot share the same name

The type

This is the type of the command, learn more

Can be used in DMs?

This says if the command can be used in user's DMs, if it is unchecked then this command can ONLY be used in servers

Ignore Bots?

True - This command cannot be triggered by any other bot

False - This command can be triggerd by any other bot

Note: Either way, the command cannot be ran by your bot
