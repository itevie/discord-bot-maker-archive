The Command Line Interface

Don't want to use the UI? Well you can do some simple things with 1 line in your terminal

At the moment, it's very basic

Note: every flag must be split by a space

Important

The CLI is NOT tested, errors may occur, use this at your own risk
This was designed as an easier, simpler alternative to coding Discord bots
Therefore, CLI was not fully thought out

Starting a bot

To start a bot, just remember it's name, open a termianl and do:

botMaker.exe --bot=bot_name

Replace "botMaker.exe" with whatever this application is called on your computer and "bot_name" with the name of your bot

For example:

botMaker.exe --bot=myAmazingBot

You can run as many bot's as you want by repeating --bot example:

botMaker.exe --bot=myAmazingBot --bot=myAmazingImageBot --bot=mee6clone

Note: do NOT ever open 2 instances of this application, it WILL break your data.

Changing where data is stored

Changing where your data is stored is useful if:

  • You want multiple instances of the app
  • Your storing the bot on another server (external hosts)

To do this use the --data flag:

botMaker --data=(directory)

But replacing (directory) with which folder you want the data to be stored in:

botMaker --data=./

This will store data.json in the same folder as the app was started in.

Force open UI

Want the UI (window) open whilst running bots using --bot?

Use --force-window, that's all

Example with all of the above:

botMaker --bot=mee6 --bot=mee7 --force-window --data=./data/
